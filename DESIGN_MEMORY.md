# Design Memory

## Brand Tone
- **Adjectives:** corporate, professional, clean
- **Avoid:** cluttered layouts, technical jargon in UI (logs)

## Layout & Spacing
- **Density:** comfortable
- **Corner radius:** 12px (corporate/professional)
- **Shadows:** subtle (0 10px 30px rgba(0,0,0,0.05))

## Typography
- **Headings:** System Sans-Serif, Extra Bold (800)
- **Body:** System Sans-Serif, Regular

## Color
- **Primary:** #ffffff (White background)
- **Accent:** #004280 (Advantech Blue)
- **Secondary Accent:** #F39800 (Advantech Orange)
- **Text:** #333333 (Dark Gray)

## Interaction Patterns
- **Feedback:** Real-time camera feed with immediate AI text updates.
